## Instructions
1. Clone the repository first and install the requirements packages :
   
   ```bash
   git clone https://github.com/mahdiadham/weather-app.git
   cd weather-app
   npm install
   ```
2. Start the project with command :
   
   ```bash
   npm start
   ```
3. Enjoy it !
